Font

There are a lot of fonts used, all from google fonts.

The way fonts were used is by adding the embeded link of the font to the html folder.
And then using the font in css folder by using "font-family" and the name of the font.

12 fonts has been used in this website so all of them would not be included, but one is showed as an example.

Name:Raleway used:shop 
        .produc2__matn {
            font-family: "Raleway", sans-serif;
        }
